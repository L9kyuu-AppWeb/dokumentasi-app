Pengguna mengakses aplikasi melalui browser. Request diproses web server dan diteruskan ke aplikasi backend. Aplikasi berkomunikasi dengan database untuk menyimpan dan mengambil data, lalu mengembalikan respons dalam bentuk HTML/JSON.

Database

Struktur Database

Perancangan database yang efisien untuk performa optimal.

📊 Gambaran Umum

  • • 10+ tabel utama
  • • Relasi One-to-Many dan Many-to-Many
  • • Normalisasi hingga 3NF
  • • Index strategis untuk query cepat

🔗 Relasi Utama

  • • Users → Links (One-to-Many)
  • • Links → Clicks (One-to-Many)
  • • Users → Roles (Many-to-One)
  • • Categories → Links (One-to-Many)

API

Dokumentasi API

API RESTful untuk integrasi dengan aplikasi pihak ketiga.

MethodURLParameterResponse
POST/api/linksurl (string), custom (string, optional)JSON — data tautan baru
GET/api/linkspage, limit, searchJSON — daftar tautan
GET/api/links/{id}—JSON — detail tautan + statistik
DELETE/api/links/{id}—JSON — konfirmasi hapus
POST/api/auth/loginemail, passwordJSON — token + user data

📤 Contoh Request

POST /api/links
{
  "url": "https://example.com/artikel/panjang",
  "custom": "artikel-1"
}

📥 Contoh Response

{
  "status": "success",
  "data": {
    "short_url": "https://ring.kr/artikel-1",
    "original_url": "https://example.com/artikel/panjang",
    "clicks": 0
  }
}
Instalasi

Langkah Instalasi

Ikuti langkah-langkah berikut untuk menjalankan aplikasi di lingkungan Anda.

1
Clone Project
git clone https://github.com/L9kyuu-AppWeb/ringkasurl.git
2
Install Dependency
composer install & npm install
3
Konfigurasi Environment
Salin .env.example ke .env, atur database
4
Migrasi Database
php artisan migrate --seed
5
Jalankan Server
php artisan serve & npm run dev
6
Verifikasi
Buka http://localhost:8000
