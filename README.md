# Dokumentasi App

Direktori terpusat untuk dokumentasi dan showcase proyek digital.

## Akses

Kunjungi **[dokumentasi-app](https://l9kyuu-appweb.github.io/dokumentasi-app/)** untuk melihat dashboard utama.

## Struktur

| Direktori | Deskripsi |
|-----------|-----------|
| `devstore/` | Showcase direktori proyek developer — impor JSON, filter tech-stack, galeri |
| `home-laragon/` | Dashboard pengganti index Laragon — akses cepat semua proyek lokal |
| `contoh-dokumentasi.md` | Template & panduan penulisan dokumentasi aplikasi |

## Cara Penggunaan Lokal

1. Clone repo ke `C:\laragon\www\dokumentasi-app`
2. Akses `http://localhost/dokumentasi-app/`
3. Pilih direktori proyek dari dashboard
