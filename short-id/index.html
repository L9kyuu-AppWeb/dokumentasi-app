Tentang short-id

Latar Belakang: Media sosial seperti Instagram, TikTok, dan Twitter hanya menyediakan satu slot tautan di bio. Pengguna kesulitan membagikan banyak tautan (portfolio, GitHub, YouTube, WhatsApp) hanya dari satu URL.

Masalah: Mengganti URL di bio setiap kali ingin mempromosikan konten baru itu merepotkan. Followers tidak bisa melihat semua tautan sekaligus.

Solusi: short-id adalah platform sederhana yang mengubah satu slot tautan menjadi halaman berisi semua tautan penting Anda — lengkap dengan analytics dan kustomisasi tema.

Target Pengguna: Kreator konten, freelancer, pebisnis online, developer, dan siapa pun yang ingin membagikan banyak tautan dari satu tempat.

Arsitektur Sistem

🌐 Browser
↓
Web Server (Apache/Nginx)
↓
Laravel 13 (Application)
↓
Livewire 4 (Reactive UI)
↓
🗄️ Database (SQLite / MySQL)

Browser mengirim request → Web Server meneruskan ke Laravel → Livewire menangani interaksi real-time tanpa reload → Data disimpan ke Database. Livewire memperbarui DOM secara selektif sehingga halaman terasa seperti SPA tanpa perlu JavaScript framework tambahan.

Database

short-id menggunakan 4 tabel utama dengan relasi sederhana.

users

Data pengguna: nama, email, username, avatar, bio, tema

links

Tautan milik user: title, url, posisi, status aktif

page_views

Riwayat kunjungan halaman publik per user

link_clicks

Riwayat klik pada setiap tautan

Relasi Utama

users 1───N links 1───N link_clicks

users 1───N page_views

Instalasi

1. Clone Project
git clone https://github.com/L9kyuu-AppWeb/short-id.git
cd short-id
2. Install Dependency
composer install
npm install
3. Konfigurasi Environment
cp .env.example .env
php artisan key:generate
4. Migrasi Database
php artisan migrate
php artisan storage:link
5. Menjalankan Server
php artisan serve
# Buka http://localhost:8000
