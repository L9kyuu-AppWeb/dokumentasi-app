Bertindaklah sebagai UI/UX Designer, Technical Writer, Product Marketing, dan Software Architect profesional.

Saya telah menyelesaikan sebuah aplikasi dan ingin membuat website dokumentasi yang bukan hanya sebagai panduan penggunaan, tetapi juga sebagai media promosi agar pengunjung tertarik menggunakan aplikasi saya.

Target dokumentasi:
- Profesional
- Modern
- Clean
- Mudah dipahami
- Meyakinkan calon pengguna
- Cocok dijadikan landing page produk

Gunakan gaya bahasa formal namun mudah dipahami.

Buat dokumentasi dengan struktur berikut.

# 1. Hero Section
Berisi:
- Nama aplikasi
- Tagline
- Deskripsi singkat
- Tombol CTA:
  - Demo
  - Dokumentasi
  - Hubungi
  - GitHub (opsional)
- Screenshot Hero

---

# 2. Tentang Aplikasi

Jelaskan:
- Latar belakang
- Permasalahan yang diselesaikan
- Tujuan aplikasi
- Target pengguna
- Manfaat utama

# 3. Keunggulan Aplikasi

Minimal 6 poin.

Setiap poin harus memiliki:
- Icon
- Judul
- Penjelasan manfaat

✔ Responsive
✔ Multi User
✔ Mudah digunakan
✔ Modern UI
