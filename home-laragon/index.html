FAQ

Pertanyaan yang paling sering ditanyakan.

Apakah Laragon Hub perlu database?
Tidak. Laragon Hub 100% berbasis filesystem — scan folder www langsung, tanpa database, tanpa migrasi, tanpa query.
Bagaimana cara install?
Clone atau download index.php ke C:\laragon\www, lalu akses http://localhost/home-laragon. Zero build step.
Apakah aplikasi ini mengganggu proyek yang sudah ada?
Tidak. Laragon Hub hanya membaca direktori — tidak memodifikasi file proyek Anda sama sekali.
Kenapa saya diarahkan ke restricted.php?
Karena diakses dari IP publik. Laragon Hub hanya bisa diakses dari localhost atau jaringan lokal, demi keamanan.
Apakah bisa diakses dari HP atau tablet?
Bisa, selama perangkat terhubung ke jaringan lokal yang sama. Grid responsif menyesuaikan ukuran layar.
Apakah ada versi untuk Mac atau Linux?
Saat ini khusus Windows + Laragon. Karena memakai PHP murni, adaptasi ke environment lain sangat memungkinkan.
Instalasi

Lima langkah, tanpa build step.

Clone project

Clone repository ke direktori proyek Anda.

cd C:\laragon\www git clone https://github.com/username/home-laragon.git

Konfigurasi Ngrok (opsional)

Install Ngrok CLI, pastikan tersedia di PATH Windows, lalu ganti variabel $url di index.php dengan URL Ngrok Anda.

Akses dashboard

Buka browser dan akses proyeknya.

http://localhost/home-laragon

Verifikasi instalasi

Pastikan dashboard muncul, search filter berfungsi, tombol aksi bekerja, dan modal PHP Info dapat dibuka.
