Tentang Aplikasi

Menyusun RPS berbasis OBE adalah tugas wajib yang memakan waktu berhari-hari. Dosen harus merumuskan CPL, CPMK, Sub-CPMK, bahan kajian, hingga kriteria penilaian — lalu menuangkannya ke dalam prompt yang tepat agar AI bisa menghasilkan RPS yang sesuai standar.

Generator Prompt RPS hadir untuk menyederhanakan proses ini. Cukup isi form terstruktur, dan aplikasi akan merangkai prompt siap pakai yang bisa langsung kamu gunakan di ChatGPT atau AI lain.

🎯

Tujuan

Mempercepat penyusunan prompt RPS berkualitas tanpa perlu menulis ulang struktur berulang kali.

👤

Target Pengguna

Dosen, Koordinator Prodi, dan Tim Kurikulum yang menyusun RPS berbasis OBE.

Penjelasan: Aplikasi berjalan sepenuhnya di browser. User mengisi form, JavaScript memproses data menjadi prompt terstruktur, lalu user menyalin prompt ke ChatGPT. Tidak ada server, database, atau API eksternal yang diperlukan.

Database

🚫

Zero Database

Generator Prompt RPS tidak menggunakan database. Semua data diproses dan ditampilkan di browser tanpa disimpan ke server mana pun. Ini berarti:

  • Privasi pengguna terjaga — tidak ada data yang keluar dari perangkat
  • Tidak perlu instalasi atau migrasi database
  • Bisa digunakan langsung tanpa backend

API

🔌

Tidak Memerlukan API

Aplikasi ini berjalan 100% di sisi klien tanpa backend atau API eksternal. Tidak ada endpoint, request, atau response — cukup buka file HTML dan gunakan.

Catatan: Arsitektur ini membuat aplikasi sangat ringan, aman, dan mudah dihosting di mana saja (termasuk GitHub Pages, Netlify, atau folder lokal).

Instalasi

📥

1. Clone / Download

git clone https://github.com/user/prompt-rps.git
cd prompt-rps

Atau download ZIP dan ekstrak ke folder lokal.

📂

2. Buka File

buka index.html
# atau double-click file

Tidak perlu server. Cukup buka langsung di browser.

🌍

3. Deploy ke Hosting (opsional)

# GitHub Pages
git push origin main

# Netlify
netlify deploy --prod --dir=.

Bisa juga upload folder ke hosting statis manapun.

FAQ — Pertanyaan Umum

Apa itu Generator Prompt RPS?
Tool web untuk membantu dosen menyusun prompt RPS berbasis OBE yang terstruktur dan siap digunakan di ChatGPT.
Apakah aplikasi ini gratis?
Ya, gratis dan open source. Bisa digunakan tanpa biaya.
Apakah perlu koneksi internet?
Tidak. Aplikasi berjalan sepenuhnya offline di browser setelah halaman dimuat.
Data saya aman?
100% aman. Semua data diproses di browser dan tidak pernah dikirim ke server manapun.
Apakah perlu install sesuatu?
Tidak. Cukup buka file HTML di browser. Tidak ada instalasi atau dependency.
Bisa digunakan di HP?
Bisa. Tampilan sudah responsif untuk layar kecil.
Bagaimana cara menyimpan data?
Saat ini data tidak disimpan. Bisa salin prompt yang dihasilkan ke notes atau file teks.
Apakah bisa export ke PDF?
Belum. Saat ini output berupa teks prompt. Fitur export sedang dalam rencana pengembangan.
Browser apa yang didukung?
Chrome, Firefox, Edge, dan browser modern lainnya. Internet Explorer tidak didukung.
Bagaimana cara berkontribusi?
Fork repository di GitHub, buat perubahan, dan kirim pull request. Semua kontribusi disambut.
Apakah bisa custom template prompt?
Belum. Template masih tetap. Fitur kustomisasi ada di roadmap v2.0.
